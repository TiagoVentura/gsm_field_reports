# GSM Field Reports

Public-facing static host for the field reports of the Global Social Media Study (Stage 2).

The reports are rendered from the private code repo at `TiagoVentura/gsm_second_stage_code` and mirrored here so we can share a stable URL with coauthors and partners without exposing the source code.

| Batch | Countries | URL |
|---|---|---|
| 1 | Australia, Denmark, India, Netherlands | [field_batch_1.html](https://TiagoVentura.github.io/gsm_field_reports/field_batch_1.html) |
| 2 | France, Turkey, UK, Latvia, Bosnia      | [field_batch_2.html](https://TiagoVentura.github.io/gsm_field_reports/field_batch_2.html) |
| 3 | Brazil, Germany, Sweden, Italy          | [field_batch_3.html](https://TiagoVentura.github.io/gsm_field_reports/field_batch_3.html) |

Index page: https://TiagoVentura.github.io/gsm_field_reports/

## Updating

Re-render the report in the source repo, copy the html into this repo's root, commit, and push. GitHub Pages will redeploy automatically.
